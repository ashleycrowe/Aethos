## 🎯 CORE PHILOSOPHY

**The Anti-Intranet Principle:**
> "Legacy intranets are where data goes to die. Aethos is where data transforms into organizational clarity."

**We are NOT:**
- A file storage system
- A collaboration platform
- A security/compliance tool (primarily)
- A traditional intranet

**We ARE:**
- An intelligence layer on top of Microsoft 365
- A metadata enrichment engine
- A bridge between siloed enterprise data
- An operational clarity platform

---

## 🧭 GUIDING PRINCIPLES

### 1. **Metadata-First Architecture**
We never store file contents - only pointers and enriched metadata. This ensures:
- Lightning-fast performance
- Privacy by design
- Scalability without storage costs

### 2. **Operational Architect, Not Security Janitor**
Language matters. We avoid "risk detected" and use "operational inefficiency identified."

Users are:
- **Architects** (not admins)
- **Curators** (not managers)
- **Operators** (not users)

### 3. **Cinematic Glassmorphism**
Beauty and functionality are not mutually exclusive. The "Aethos Glass" aesthetic:
- Deep space background (#0B0F19)
- Starlight cyan accents (#00F0FF)
- Supernova orange alerts (#FF5733)
- 70-95% opacity glass cards with heavy blur

### 4. **Tiered Intelligence**
- **Base Tier ($499/mo):** Metadata intelligence for all
- **AI+ Tier ($698/mo):** Content reading for semantic search

No free tier. 14-day trial with 1 workspace limit.

## 📜 BRAND VOICE

**Tone:**
- Confident, not arrogant
- Technical, but accessible
- Empowering, not condescending

**Terminology:**
- "Workspace" not "Hub" (avoid silo implication)
- "Communication Bridge" not "Portal"
- "Intelligence Layer" not "Dashboard"
- "Operational Clarity" not "Security Posture"

**Forbidden Terms:**
- "Janitor", "Cleanup", "Housekeeping"
- "Risk Detected" (use "Inefficiency Identified")
- "Admin Panel" (use "Operational Command Center")

## 🎨 DESIGN PHILOSOPHY

**The Aethos Glass Aesthetic:**
- Cinematic, not corporate
- Spacious, not cramped
- Intelligent, not cluttered
- Glassmorphic, not flat/skeuomorphic

**Typography:**
- UPPERCASE TRACKING FOR LABELS
- Title Case for headings
- Sentence case for body text
- Monospace for data/metrics

**Motion:**
- Subtle fade-ins (300ms)
- GPU-accelerated transforms only
- Respect `prefers-reduced-motion`
