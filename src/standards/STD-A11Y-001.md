## 🌌 The Aethos Glass Challenge
Aethos uses a cinematic glassmorphism aesthetic. Standard glass techniques often fail WCAG 2.1 AA contrast requirements. This standard mandates specific adaptations to ensure Aethos is accessible to all users while maintaining its "Deep Space" aesthetic.

---

## 🚨 MANDATORY CRITICAL RULES

1.  **GLASS CONTRAST (4.5:1):** Any surface containing body text MUST have an underlying semi-opaque layer (minimum `bg-black/40`) to ensure text remains readable against dynamic background gradients.
2.  **CYAN FOCUS GLOW:** Default browser focus rings are forbidden. All interactive elements must use a `ring-2 ring-[#00F0FF] ring-offset-2 ring-offset-[#0B0F19]` style.
3.  **SIDECAR LIVE REGIONS:** All asynchronous metadata updates (Nexus Sync, Flashlight Scans) MUST use `aria-live="polite"`.
4.  **KEYBOARD ESCAPE:** All drawers (Inspection Drawer, Demo Lab) MUST close on `Escape` and trap focus within the drawer.
5.  **SCREEN READER LABELS:** Icon-only buttons (Trash, Archive, Search) MUST have `aria-label` and `title` attributes.

## ⌨️ Keyboard Navigation Patterns

### 1. The Bulk Actions Toolbar
When the `BulkActionsToolbar` appears, it is a "floating" context. 
- **Requirement:** It must be reachable via a single `Tab` from the site list.
- **Implementation:** Use a `role="toolbar"` and ensure buttons have descriptive `aria-label`s.

### 2. Focus Trapping in Drawers
```tsx
// MANDATORY: Drawer Focus Trap
useEffect(() => {
  if (isOpen) {
    document.body.style.overflow = 'hidden';
    const focusable = drawerRef.current?.querySelectorAll('button, [href], input');
    (focusable?.[0] as HTMLElement)?.focus();
  }
}, [isOpen]);
```

## 🔊 Screen Reader (ARIA) Guidelines

### Entity Pointers (The Sidecar Lens)
Since we deal with metadata pointers, ARIA labels must be explicit about the resource location.
- **Correct:** `aria-label="SharePoint Site: Marketing Legacy, 142 Gigabytes, Inactive for 2 years"`
- **Incorrect:** `aria-label="Marketing Legacy"`

### Status Announcements
Use `role="status"` for the `NexusSync` component:
```tsx
<div role="status" aria-live="polite">
  {status === 'saving' && "Syncing metadata to Cosmos DB..."}
  {status === 'saved' && "Metadata pointer secured."}
</div>
```

## ✅ Compliance Checklist (AppSource Ready)

- [ ] **Skip Link:** "Skip to Main Content" link exists and targets the `<main>` element.
- [ ] **Keyboard Nav:** Can complete a "Nexus Workspace" creation using only a keyboard.
- [ ] **Alt Text:** All branding SVG icons have `aria-hidden="true"` if decorative, or `role="img"` if functional.
- [ ] **Motion:** All animations respect `prefers-reduced-motion` media queries.
- [ ] **Tab Order:** Navigation follows the visual hierarchy (Sidebar -> Header -> Content -> Bulk Toolbar).
