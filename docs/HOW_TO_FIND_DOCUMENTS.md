# How to Find Documents in the Document Control System

## 🎯 Quick Answer

**To find documents:**
1. Click **"Documents"** in the sidebar (under Tools section)
2. Click the **"All Documents (8)"** tab
3. Browse all 8 demo documents!

---

## 📍 Step-by-Step Navigation

### **Step 1: Access Document Control**
- Look at the **left sidebar** in your Aethos app
- Find the **"Tools"** section
- Click **"Documents"** (📋 FileCheck icon)

### **Step 2: Choose Your View**
You'll see **3 tabs** at the top:

#### **📚 Libraries Tab** (Default)
- Shows 4 document libraries
- Each library shows:
  - Library name & description
  - Compliance standard badge
  - Document count
  - Numbering prefix

#### **📄 All Documents (8) Tab** ← **Click This!**
- Shows ALL 8 documents in a grid
- Each document card displays:
  - Document number & title
  - Health score (with color indicator)
  - Status badge (Published, In Review, Expired, etc.)
  - Last modified date
  - Author
  - Document type
  - Compliance standard

#### **📊 Analytics Tab**
- Health score distribution
- Document status breakdown
- Top contributors
- Recent activity

## 🎨 Health Score Color Guide

Documents are color-coded by health:
- 🟢 **Emerald (90-100%)**: Excellent - Document is in great shape
- 🔵 **Cyan (75-89%)**: Good - Minor improvements needed
- 🟡 **Amber (60-74%)**: Fair - Some attention required
- 🟠 **Orange (40-59%)**: Poor - Needs significant work
- 🔴 **Red (0-39%)**: Critical - Urgent action required

---

## 🏷️ Status Badges

- 📝 **Draft**: Being written
- ⏳ **In Review**: Under approval
- ✅ **Approved**: Approved but not published
- 📋 **Published**: Active and accessible
- 📦 **Archived**: Retired but kept for records
- 🔴 **Expired**: Needs renewal

---

## 💡 Pro Tips

### **Filter by Health**
Documents are sorted by default, but you can visually scan:
- Look for 🟢 green scores for healthy docs
- Look for 🔴 red scores for docs needing attention

### **Filter by Status**
- All **Published** docs have ✅ badge
- All **Expired** docs have 🔴 badge
- All **In Review** docs have ⏳ badge

### **Filter by Library**
Each document card shows its library name:
- **HR Policies & Procedures** - Employee-related
- **IT Security & Governance** - Security & tech
- **Quality Management System** - QMS processes

### **Filter by Compliance**
Look for the shield icon:
- 🛡️ **ISO 9001** - Quality management
- 🛡️ **SOC 2** - Security & availability

## 🔧 Searching for Specific Documents

### **By Number**
Look for the document number prefix:
- **POL-** = Policy
- **SOP-** = Standard Operating Procedure
- **STD-** = Standard
- **CHK-** = Checklist
- **PLN-** = Plan
- **MAT-** = Matrix

### **By Topic**
Read the title on each card:
- HR topics: Remote Work, Performance Review, Onboarding
- IT topics: Data Security, Password Management, Incident Response, Access Control
- QMS topics: Document Control
